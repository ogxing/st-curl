# SillyTavern `/curl` Extension

This extension adds a `/curl` slash command to SillyTavern.

## Usage

In SillyTavern, send:

    /curl http://localhost:5566/toggle

It will call the URL and complete silently.

## Install

Place the `st-extension` folder into your SillyTavern extensions directory, or upload it to GitHub and install it as a third-party extension.
